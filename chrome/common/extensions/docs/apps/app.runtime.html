Events

onLaunched

chrome.app.runtime.onLaunched.addListener(function(object launchData) {...});

Listener parameters

launchData
( optional object )
Undocumented.
intent
( object )
Undocumented.
action
( string )
The WebIntent being invoked.
data
( any )
Data associated with the intent.
postFailure
( function )
Callback to be compatible with WebIntents.
postResult
( function )
Callback to be compatible with WebIntents.
type
( string )
The MIME type of the data.
