API reference: chrome.experimental.alarms

Methods

clear

chrome.experimental.alarms.clear(string name)

Clears the alarm with the given name.

Parameters

name
( optional string )
The name of the alarm to clear. Defaults to the empty string.

clearAll

chrome.experimental.alarms.clearAll()

Clears all alarms.

create

chrome.experimental.alarms.create(string name, experimental.alarms.AlarmCreateInfo alarmInfo)

Creates an alarm. After the delay is elapsed, the onAlarm event is fired. If there is another alarm with the same name (or no name if none is specified), it will be cancelled and replaced by this alarm. string.

Parameters

name
( optional string )
Optional name to identify this alarm. Defaults to the empty
alarmInfo
( experimental.alarms.AlarmCreateInfo )
Undocumented.

get

chrome.experimental.alarms.get(string name, function AlarmCallback)

Retrieves details about the specified alarm.

Parameters

name
( optional string )
The name of the alarm to get. Defaults to the empty string.
AlarmCallback
( function )
Undocumented.
Parameters
alarm
( experimental.alarms.Alarm )
Undocumented.

Callback function

The callback parameter should specify a function that looks like this: 

function(experimental.alarms.Alarm alarm) {...};
alarm
( experimental.alarms.Alarm )
Undocumented.

getAll

chrome.experimental.alarms.getAll(function AlarmListCallback)

Gets an array of all the alarms.

Parameters

AlarmListCallback
( function )
Undocumented.
Parameters
alarms
( array of experimental.alarms.Alarm )
Undocumented.

Callback function

The callback parameter should specify a function that looks like this: 

function(array of experimental.alarms.Alarm alarms) {...};
alarms
( array of experimental.alarms.Alarm )
Undocumented.

Events

onAlarm

chrome.experimental.alarms.onAlarm.addListener(function(experimental.alarms.Alarm alarm) {...});

Fired when an alarm has elapsed. Useful for transient background pages.

Listener parameters

alarm
( experimental.alarms.Alarm )
The alarm that has elapsed.

Types

experimental.alarms.Alarm

( object )
Undocumented.
delayInMinutes
( number )
Original length of time in minutes after which the onAlarm event should fire. TODO: need minimum=0
name
( string )
Name of this alarm.
repeating
( boolean )
True if the alarm repeatedly fires at regular intervals, false if it only fires once.

experimental.alarms.AlarmCreateInfo

( object )
Undocumented.
delayInMinutes
( number )
Length of time in minutes after which the onAlarm event should fire. Note that granularity is not guaranteed: this value is more of a hint to the browser. For performance reasons, alarms may be delayed an arbitrary amount of time before firing. TODO: need minimum=0
repeating
( optional boolean )
True if the alarm should repeatedly fire at regular intervals. Defaults to false.
