How to use these builds

WARNING: All builds posted here are experimental! They have not been vetted by Google, me, nor anyone else for security or stability, and under no circumstances should you use them as your day-to-day browser! They are provided for testing of the experimental WebVR APIs only.

In order to enable access to the WebVR APIs in these builds you must select the "Enabled" option from the drop-down menu for the "Enable WebVR" flag (enter chrome://flags/#enable-webvr in the URL bar) and the "Gamepad Extensions" flag (enter chrome://flags/#enable-gamepad-extensions in the URL bar), or launch Chromium from the command line with the --enable-webvr and --enable-gamepad-extensions flags.
