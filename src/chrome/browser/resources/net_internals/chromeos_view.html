Store system logs

This stores system_logs_{date}-{time}.txt.zip to the Downloads directory. The archive contains a single file identical to the system_logs file attached to Feedback reports.

Also writes a policies_{date}-{time}.json file for providing policy configurations when that might be a factor.

Prefer this when uploading logs to issues or sending logs to developers (unless otherwise requested).

Store debugd system logs

This stores the logs collected by debugd as individual files in a single archive file in the Downloads directory: debug-logs_{date}-{time}.tgz.

Also writes a policies_{date}-{time}.json file for providing policy configurations when that might be a factor.

Identical to the above, but also incldes the Chrome user logs. The output file is named combined-logs_{date}-{time}.tgz.
